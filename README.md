# 💰 Financial KPI Dashboard (Power BI)

## 📌 Objective

This project focuses on building an **interactive Power BI dashboard** to track key financial metrics such as:

* Revenue trends
* Budget vs Actual performance
* Cash flow analysis

The dashboard includes **filters and drill-down capabilities** to enable deeper analysis.
📁 Deliverable: `.pbix` file

---

## 📊 Problem Statement

Organizations often lack **integrated and interactive financial reporting**, which limits visibility into:

* Revenue performance
* Profitability
* Cash flow trends

This project addresses the gap by developing a **dynamic dashboard** with:

* Time-based analysis
* Interactive filtering
* Drill-down insights
