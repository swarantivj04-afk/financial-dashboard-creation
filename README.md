##Objective;
This project involves building an interactive Power BI dashboard to track financial KPIs, including revenue trends, budget vs actuals, and cash flow, with filters and drill-down capabilities. Deliverables includes a .pbix file 
##problem statement: 
  Lack of integrated and interactive financial reporting limits visibility into revenue, profitability, and cash flow performance. This project addresses the gap by building a dynamic dashboard with time-based analysis, filtering, and drill-down capabilities to generate actionable insights.

    daraanalyst case study is attached in the repository

      Visuals: Revenue & Profit Trend, Budget vs Actual variance %, Product/Service performance, Receivables Aging, Cash Flow Waterfall
